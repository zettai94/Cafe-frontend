# Indie Bites Café

A modern, responsive landing page for a fictional café showcasing clean UI/UX design and smooth user experience.

🔗 **[View Live Site](https://indiebitescafe.vercel.app/)**

## About
![Indie Bites Front End](.src/assets/images/IndieBitesResponsive.png)
This project demonstrates front-end web development skills including responsive design, modern CSS techniques, and user-friendly navigation. Built with a mobile-first approach to ensure optimal viewing across all devices.

## Technologies Used
- **React.js**: Component-based UI architecture.
- **React Router**: Client-side routing for seamless page transitions.
- **CSS3/Bootstrap**: Responsive grid systems and custom Flexbox layouts.
- **EmailJS**: Serverless contact form integration.
- **Vercel**: Continuous deployment and hosting.

## Features
- **Fully Responsive**: Fluid transitions across mobile, tablet, and desktop.
- **Retractable Navigation**: Optimized screen real estate for ordering workflows.
- **Optimized Performance**: Lazy-loaded assets and clean DOM structure.
- **Interactive Menu**: Dynamic category filtering for products.
